# Camera Technical Specifications Template

This template defines the standard set of attributes (attributeName) for camera products in the TUNG Shop system. Use these names consistently to ensure uniform display across the product detail pages.

- **Loại pin**: (e.g., NP-FZ100 Lithium-Ion)
- **Thời lượng pin**: (e.g., Khoảng 750 lần chụp)
- **Chất liệu thân máy**: (e.g., Hợp kim magiê)
- **Kích thước (W x H x D)**: (e.g., 130,3 x 96,4 x 72,3 mm)
- **Trọng lượng**: (e.g., 610 g (Chỉ thân máy), 695 g (Có pin, thẻ nhớ))
- **Nhiệt độ hoạt động**: (e.g., 0 đến 40°C)

## 2. Imaging (Hình ảnh)
- **Ngàm ống kính**: (e.g., Sony E)
- **Loại cảm biến**: (e.g., CMOS Full-Frame)
- **Độ phân giải cảm biến**: (e.g., 33 Megapixel hiệu dụng)
- **Chế độ chống rung ảnh**: (e.g., Chống rung 5 trục)
- **Tỉ lệ ảnh**: (e.g., 3:2, 16:9)
- **Định dạng ảnh**: (e.g., RAW, JPEG, HEIF)
- **Độ sâu màu**: (e.g., 14-Bit)

## 3. Storage & Connectivity (Lưu trữ và kết nối)
- **Khe cắm thẻ nhớ**: (e.g., Khe 1: CFexpress Type A; Khe 2: SD UHS-II)

- **Audio I/O**: (e.g., 1x 3.5 mm Headphone; 1x 3.5 mm Mic)
- **Power/Other I/O**: (e.g., 1x USB-C Input; 1x USB-C Data)

## 3. Flash System (Hệ thống đèn Flash)
- **Đèn Flash trong**: (e.g., Không có)

## 4. Focus mechanics (Cơ chế lấy nét)
- **Cơ chế lấy nét**: (e.g., Lấy nét tự động và thủ công)

## 5. Display & Viewfinder (Màn hình & Kính ngắm)
